9.5 Flex弹性布局

1. Flex

1 . main-axis(主轴)

2. cross-axis(交叉轴)

3. flex-direction:

  • Flexbox Container

    Attribute Description
    row main-axis(主轴)是 row
    column main-axis(主轴)是 column
    row-reverse main-axis(主轴)是 row,逆序排列
    column-reverse main-axis(主轴)是 column,逆序排列

4. justify-content 和 align-items

  • justify-content ： 主轴排列方向

  • align-items：交叉轴排列方向

    Attribute Description
    center
    flex-start
    flex-end

5. flex-wrap

  • 定义是否换行

    Attribute Description
    nowrap 不换行
    wrape 换行

6. flex-flow

  • flex-direction、jflex-wrap 的简写

7. align-content

  • 当 flex-wrape：wrape 时，生效；用于行于行之间的对齐方式

    Attribute Description
    center
    flex-start
    flex-end
    initial
    space-around
    space-evenly
    space-between

2. Flexbox Item

1. order

  • 默认值： 0
  • 数值越大越排在后
  • 可以取负值

2. align-self

  • 可以用于改写 flex container 中，align-items 中的设定值
  • 属性值与 flex container 相同

3. flex-basis: 好像没有效果

  • 是设定 flex-item 的主轴方向上的大小

    Attribute Description
    0 width: 0px;
    100px 原值 + 100px
    auto 原值

4. flex-grow

当主轴方向有剩余空间时，flex-item 沿主轴方向扩展的设定,flex-grow: 0；表示保持原始值，不放大；

5. flex-shrink

当主轴方向空间不够时，flex-item 沿主轴方向缩小的设定,flex-shrink: 0；表示保持原始值；

6. Flex

Flex Item 项目中： flex-grow + flex-shrink + flex-basis 的简写形式
