6.17 边框与背景

1 应用边框样式

1. 定义边框宽度

  • 表 1 基本边框属性

    Attribute Description
    border-width 设置边框宽度
    border-style 设置边框使用样式
    border-color 设置边框颜色

border-width 的取值

  1. 百分比
  2. px , em rem
  3. thin 、medium、thick

2. 定义边框样式

border-style 取值

  1. none
  2. dashed
  3. dotted
  4. double
  5. groove： 槽线式边框
  6. inset：具有内嵌效果的边框
  7. outset：具有外嵌效果的边框
  8. ridge ：脊线边框
  9. solid：实线边框

  • 表 5

    Attribute Description Value
    border-image-source 设置图像来源 none 或 url

    border-image-width 边框宽度 auto、1 ～ 4 长度值或百分比
    border-image-outset 边框向外扩展部分 1 ～ 4 长度值或百分比
    border-image-repeat 边框区域填充模式 stretch、repeat、space 和 round
