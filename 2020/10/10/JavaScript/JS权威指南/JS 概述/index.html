第一章 1 JS概述

JavaScript 概述

  1. JavaScript 是面向 Web 的编程语言，所有的现代浏览器—基于桌面系统、游戏机、平板电脑和智能手机均包含了 JavaScript 解析器；
  2. JavaScript 的语法源自 Java
  3. JavaScript 的一等函数(first-class-function)来自与 Scheme
  4. JavaScript 的基于原型（prototype-base)的继承来自于 Self
  5. 任何一种编程语言都有各自的开发平台、标准库或 API 函数；
  6. JavaScript 包括：
    • JavaScript 语言核心： ECMAScript
    • 客户端 JavaScript： DOM，BOM
  7. JavaScript 输入输出功能(类似网络、存储和图形相关的复杂特性)是由其所属的宿主环境(host enviroment)提供-浏览器
  8. JavaScript 语言核心针对文本，数组，日期和正则表达式的操作定义提供了很少的 API，这些 API 不包括输入，输出功能；
  9. 输入，输出功能是由 JavaScript 所属宿主环境（host enviroment）提供的。例如：
    • Rhio 是基于 Java 的 JavaScript 引擎，实现了通过 JavaScript 方位 Java API；
    • Node 是 Google V8 JavaScript 引擎，它的底层绑定了 POSIIX (Unix) API,包括文件， 进程， 流和套接字等，并侧重于异步 I/O,网络和 HTTP

JavaScript 语言核心

  1. ECMAScript 包括： ES 5, ES6, ES7, ES8, ES9, ES10；
  2. ES5 是语言核心基础， 要求掌握 ES6
  3. ES5 是语言核心基础：
    • 词法结构
    • 数据类型
    • 变量
    • 运算符
    • 表达式
    • 语句
    • 对象
    • 函数
    • 数组
    • 类
    • 模块
  4. ES6：
    • 变量的解构赋值
    • 字符串的扩展
    • 正则的扩展
    • 数值的扩展
    • 函数的扩展
    • 数组的扩展
    • 对象的扩展
    • 对象的新增方法
    • 字符串的新增方法
    • Symbol
    • Set 和 Map 数据结构
    • Proxy
    • Reflect
    • Promise 对象
    • Iterator 和 for …of 循环
    • Generator 函数语法
    • Generator 函数的异步应用
    • async 函数
    • Class 的基本语法
    • Class 的继承
    • Module 的语法
    • Module 的加载实现
    • 编程风格
    • 异步遍历器
    • ArrayBuffer
    • Decorator
    • 参考链接

客户端 JavaScript

  1. 包括： DOM， BOM；
    • DOM： JavaScript 操作 HTML 元素，操作 CSS 样式，事件处理
    • BOM： JavaScript 实现对 window 对象的操作

服务器端 JavaScript

  1. Rhio 是基于 Java 的 JavaScript 引擎，实现了通过 JavaScript 方位 Java API
  2. Rhino 程序能使用 Java 的 Swing 框架创建图形 UI
  3. 而 Node 是 Google V8 JavaScript 引擎，它的底层绑定了 POSIIX (Unix) API,包括文件， 进程， 流和套接字等，并侧重于异步 I/O,网络和 HTTP

JavaScript HTML5 API

  1. 包括：
    • Canvas
    • 多媒体播放
    • 本地存储
    • 离线应用程序
    • 通信 API
    • 获取地理位置信息
    • Web Workers 处理线程
    • 文件 API
