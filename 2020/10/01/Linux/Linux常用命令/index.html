1. 文件管理

1.1 cd 命令

  • cd : change direction

  • cd 切换到根目录

  • cd 切换到 Home 目录

  • cd 切换到上一级目录

  • cd 切换到/etc/apt 目录

1.2 ls 命令

  1. 语法格式： ls [选项] [文件]

  2. ls: list

  3. 常用参数：

  4. 名称 说明
    -a 显示所有文件及目录 (包括以“.”开头的隐藏文件)
    -l 使用长格式列出文件及目录信息
    -r 将文件以相反次序显示(默认依英文字母次序)
    -t 根据最后的修改时间排序
    -A 同 -a ，但不列出 “.” (当前目录) 及 “..” (父目录)
    -S 根据文件大小排序
    -R 递归列出所有子目录

1.3 参考实例

  • ls 显示根目录下的所有文件及目录

  • ls 显示 /etc/apt 路径下，所有文件及目录的详细信息

  • ls 显示当前路径下，所有文件及目录的详细信息

  • ls 显示当前路径下（包括隐藏目录）所有文件及目录的详细信息

  • ls 显示以”s”开头的文件及目录详细信息

  • ls 显示/bin 目录下的所有目录及文件的详细信息

1.4 mkdir 命令

  1. 语法格式 : mkdir [参数] [目录]

  2. 常用参数：

    名称 说明
    -m 建立目录的同时设置目录的权限
    -z 设置安全上下文
    -v 显示目录的创建过程
    -p 递归创建多级目录

  3. 参考实例

    • 创建一个名称：project and demo 目录

    • 创建一个名称：dir 目录， 并设置权限，他人不能访问

    • 递归创建目录
