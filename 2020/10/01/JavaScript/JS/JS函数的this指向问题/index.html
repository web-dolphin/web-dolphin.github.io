JS函数中this指向问题

JS 函数中的 this 指向

  1. JS 中函数的 this 指向，不是由函数定义决定的，而是由函数的调用对象决定，谁调用，就指向谁；
  2. 在嵌套函数中，通常指向其上一级对象；
  3. 在上级对象没有调用时，函数 this 指向会继续向上级对象寻找；
  4. 如果所有上级对象都不存在调用者时，this 指向 window

构造函数中的 this 指向

  1. 案例九

    • 构造函数的 this 指向： 由于在构造函数实例化过程中，new 关键字的影响，将 this 指向了 object;

return 对 this 指向的影响

  • 如果返回值是一个对象，那么 this 指向的就是返回的对象
  • 如果返回值不是一个对象，那么 this 还是指向函数的实例

严格模式 this 指向

  1. 在严格版中的默认的 this 不再是 window，而是 undefined

为什么构造函数 this 指向实例对象

  • this 指向实例对象；
  • 首先 new 关键字会创建一个空的对象；
  • 然后会自动调用一个函数 apply 方法，将 this 指向这个空对象，
  • 这样函数内部的 this 就会被这个空的对象替代；
