3.1 盒子模型

2. 盒子模型(box model)：

每一个元素在页面中都会占据一定的区域，该区域就称为元素的盒子(box),

  1. 说明

  1. Box 组成：

Content(内容)、border(边框)、padding(内边距)，margin(外边距)

  1. standard(标准) box

  1. IE box

为了解决两者定义之间的区别，通常规定： border-size 属性(attribute)

  1. 表 2 盒子模型属性
Attribute(属性) Description(说明)
box-sizing: border-box height / width = content + padding + border
box-sizing: content-box height / width = content
box-sizing: inherit 继承父级元素属性值

3. Block Container(容器):

  • 通常将父子元素关系中父元素盒子称之为容器

4. 外边距塌陷

  1. 外边距塌陷(margin collapse)：
    • 块元素的外边距有时会合并为单个外边距，这样的现象称之为外边距塌陷。
  2. 父子元素盒子模型塌陷： 合并后最大的外边距
  3. 兄弟元素间外边距塌陷： 合并后最大的外边距
