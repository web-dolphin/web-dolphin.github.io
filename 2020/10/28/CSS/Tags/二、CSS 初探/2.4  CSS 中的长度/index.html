2.4 CSS 中的长度

1. 绝对长度： cm（厘米），pt （磅）

2. 相对单位：

  • 与元素字号挂钩的单位： em;

  • 与元素字体 x 高度挂钩的单位： ex， 即字体基线到中线之间的距离，通常 1 ex = 0.5em;

  • 与根元素(html 元素)挂钩的单位： rem;

  • 与像素挂钩的单位： px；

  • 与元素属性的值得百分比挂钩的单位： %;
