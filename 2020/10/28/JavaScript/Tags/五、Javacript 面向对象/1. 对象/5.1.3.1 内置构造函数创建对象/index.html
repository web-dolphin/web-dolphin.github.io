5.1.3.1 内置构造函数创建对象

1. 文档目录

1. Object

  • 原始方式创建对象，属性和方法可以在对象创建后动态定义；

  • new Object()等同于对象字面量{}；

  • 通过new关键字，创建实例化Object对象；

  • 当Object实例化对象没有参数时，可以省略括号；

  • 使用同一接口创建多个对象时，会产生大量重复代码;

  • [2.11.9 Object内置对象]

2. Function

  • 任何函数都是由内置Function构造函数创建的；
