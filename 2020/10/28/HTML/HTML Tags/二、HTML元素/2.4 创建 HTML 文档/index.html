2.4 创建HTML文档

1. 元素介绍

1. 元素汇总表

  1. 创建 HTML 文档元素： 是指构建网页文档的基本组成元素
  • 包括：
Element(元素) Description(说明)
！DOCTYPE 定义文档规范采用的是 HTML5
html 定义 html 标记的开始
head 表示文档中原数据部分的开始
body 表示文档主体内容的开始
title 文档标题
base 设置文档中相对与 URL 解析基础的 URL
meta 添加对 HTML 文档所含数据的说明
带 charset 属性的 meta 声明 HTML 文档的字符编码
style 定义文档内嵌样式
link 载入包括样式表和网站标志在内的外部资源
带 http-equiv 属性的 meta 设置 HTML 文档的默认样式表或周期性刷新页面内容
带 rel=”prefetch”属性的 link 预先载入预计马上就会用到的资源
script 定义文档内嵌脚本
带 src 属性的 script 载入外部脚本
带 async 或 defer 属性的 script 控制外部脚本的执行时机或执行方式
noscript 显示浏览器不支持或禁用时 JavaScript 准备的内容提示信息
