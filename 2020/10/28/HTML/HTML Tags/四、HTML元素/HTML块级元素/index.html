HTML块级元素

1. 块级元素(Block)的定义

  1. 总是在新行上开始
  2. 高度，行高以及外边距和内边距都可控制；
  3. 宽度缺省是它的容器的 100%，除非设定一个宽度；
  4. 它可以容纳内联元素和其他块元素；
  5. 多个块状元素标签写在一起，默认排列方式为从上至下；
