VS Code

VS Code开发常用工具

1. Settings Sync

  1. 登陆

    • 点击左下角 `Setting``

    • ``Turn On Settings Sync`

    • LOGIN WITH GITHUB`

    • 填写邮箱，密码， 打开邮箱提取验证码

2. Faker

  • Faker 可以随机生成姓名、地址、图像、电话号码，或者经典的乱数假文段落，

3. CSS Peek

  • 可以追踪至样式表中 CSS 类和 ids 定义的地方;

  • 当你在 HTML 文件中右键，单击选择Go to Definition 或者 Peek definition 选项，它便会给你发送样式设置的 CSS 代码

4. HTML Boilerplate

  • 使用 HTML 模版插件，你就摆脱了为 HTML 新文件重新编写头部和正文标签的苦恼；

  • 你只需在空文件中输入 html，并按 Tab 键，即可生成干净的文档结构

5. Color Info

  • 获取颜色相关信息

6. SVG Viewer

  • SVG图片预览插件

  • 此插件在 Visual Studio 代码中添加了许多实用的 SVG 程序;

  • 你无需离开编辑器，便可以打开 SVG 文件并查看它们;

  • 同时，它还包含了用于转换为 PNG 格式和生成数据 URI 模式的选项

7. TODO 注释

  • 这个插件能够在你的代码中标记出所有的 TODO 注释，以便更容易追踪任何未完成的业务；

  • 在默认的情况下，它会查找 TODO 和 FIXME 关键字。当然，你也可以添加自定义表达式。

8. Icon Fonts

  • 这是一个能够在项目中添加图标字体的插件;

  • 该插件支持超过 20 个热门的图标集;

  • 包括了 Font Awesome、Ionicons、Glyphicons 和 Material Design Icons。

8. Quokka

  • Quokka 是一个调试工具插件，能够根据你正在编写的代码提供实时反馈;

  • 它易于配置，并能够预览变量的函数和计算值结果;

  • 另外，在使用 JSX 或 TypeScript 项目中，它能够开箱即用

9. Regex Previewer

  • 这是一个用于实时测试正则表达式的实用工具；

  • 它可以将正则表达式模式应用在任何打开的文件上，并高亮所有的匹配项。

10. Change Cas

  • 虽然 VSCode 内置了开箱即用的文本转换选项，但其只能进行文本大小写的转换；

  • 此插件则添加了用于修改文本的更多命名格式，包括驼峰命名、下划线分隔命名，snake_case 命名以及 CONST_CAS 命名等。

11. Minify

  • 用于压缩合并 JavaScript和 CSS 文件的应用程序；

  • 提供了大量自定义的设置，以及自动压缩保存并导出为.min 文件的选项；

  • 能够分别通过 uglify-js、clean-css和 html-minifier，与 JavaScript、CSS和 HTML 协同工作。
