3.1 HTML5 新增元素

1. HTML5 新增元素

  1. HTML5 定义的许多元素 HTML4 中也存在，但是，元素的含义已经发生了很大的变化，或者，其使用方式已经不同；
  2. HTML5 本身还新增了许多新的元素；

  • HTML5 元素主要变化： 将元素语义与其内容呈现结果的影响分开；

  • HTML元素负责文档内容的结构和含义，而内容的呈现则由 CSS 样式控制；

  • HTML5 中新增的大多数元素都具有含义；

    element discription type

    nav 导航

    footer 页脚
    details 元素的细节
    summary / details 元素可见的标题
    dialog 对话框窗口
    dir 目录列表
    menu 命令的菜单,列表
    menuitem 菜单项目
    command 命令按钮
    fieldset 围绕元素的边框(可用于表单内元素分组)
    legend 在边框上的标题

    option 选择列表选项(也可做 datalist 选项)

3.6 HTML5全局属性

1.全局属性： global attribute；

  • style：css 行内样式属性,短语元素

  • lang： 说明元素内容使用的语言,短语元素

  • id： css 样式 id，同一个页面只有一个 id,短语元素

  • class： css 类样式,短语元素

  • accesskey： input 设定一个或几个页面元素的快捷键,短语元素

  • dir：规定元素了文字的方向,短语元素

  • contenteditable：规定元素内容是可编辑,短语元素

  • draggable： 短语元素 规定了元素可以拖放,短语元素

  • dropzone 短语元素 与 draggable 配套,短语元素

    [详见拖放]

  • hidden 短语元素 隐藏元素,短语元素

  • spellcheck 短语元素 是否对元素内容做拼写检查,短语元素

  • tabindex: 短语元素 键盘 Tab 在 HTML 页面元素之间切换

    • 按下 Tab 键时，按数字的从小到大一次选取元素内容
    • 数值为负值时，不被选中

  • title 短语元素 提供元素的额外信息
