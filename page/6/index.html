2.11 HTML特殊字体

  • HTML 特殊字体： HTML 预留字符

  • 在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签；

  • 如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用字符特殊字体（character entities）
