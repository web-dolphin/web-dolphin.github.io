  • 新建一个目录，将其初始化为Git代码库

  • 在现有分支与指定的远程分支之间，建立追踪关系

  • 合并指定分支到当前分支

  • 选择一个commit，合并进当前分支

  • 删除分支

  • 删除远程分支

  • 显示有变更的文件

  • 显示当前分支的版本历史

  • 显示某个commit之后的所有变动，其”提交说明”必须符合搜索条件

  • 显示某个文件的版本历史，包括文件改名

  • 显示工作区与当前分支最新commit之间的差异

  • 显示两次提交之间的差异

  • 显示今天你写了多少行代码

  • 显示某次提交发生变化的文件

  • 显示某次提交时，某个文件的内容
