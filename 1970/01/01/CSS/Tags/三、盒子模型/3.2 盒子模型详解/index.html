3.2 盒子模型详解

1. 盒子模型

1. 创建项目

  2. 盒子的属性有四种：

  • content: 盒子

  • padding: 内边距

  • margin: 外边距

  • border: 边框

  • HTML 代码

  • 我们先将.container和.inner分别定义宽高相等的正方形，并且居中： 600px 和 400px

  • CSS 代码

  • 打开 chrome 浏览器开发者工具，查看盒子模型，此时的宽高就属于content部分: 60x60

  1. padding 属性的定义

  • 此时的盒子尺寸就与设计尺寸产生出入，为了解决冲突，就必须修改content尺寸： 20px x 20px

  1. box-sizing: border-box定义，自动计算盒子尺寸，使之为设计尺寸；

  • 回到 1. padding 属性的定义，添加box-sizing: border-box

  • 此时，盒子将重新自动计算尺寸，以维持盒子 60px x 60px 的要求

  1. 例外

  • box-sizing: border-box定义盒子尺寸为定义尺寸，当 padding 值超过盒子定义宽高时，content取之为 0,盒子实际尺寸将由 padding 值决定

  • 打开开发者工具，查看盒子尺寸

3. border 属性

  1. border: 盒子边框

  • 盒子border与content,padding共同决定的盒子的尺寸；

  • 为了维护盒子的设计尺寸，需要修改content值；

  1. border的四个扩展属性

  • border-left

  • border-right

  • border-top

  • border-bottom

  1. border属性不影响content的取之尺寸

4. margin 属性

  • 盒子的margin属性称为： 外边距，指盒子与盒子之间的距离；

  • 子盒子的外边距

  • 此时的外边距设置，right和bottom都由浏览器计算，为 auto，因此即使设置也无效；
